# 🌐 Radar Global

Um site crítico-reflexivo desenvolvido para a disciplina de **Estudos Contemporâneos** (UNAERP - 2026/01). O projeto explora e conceitua temas vitais que moldam o mundo atual, como Globalização, Pobreza, Corrupção e Sustentabilidade.

## Conteúdo do Projeto

O site aborda as seguintes temáticas através de uma navegação interativa e design moderno:
- **Conceitos Fundamentais:** Definições claras sobre Globalização, Nova Ordem Mundial, Corrupção, Políticas Públicas, Estado/Governo, Pobreza e Sustentabilidade.
- **Casos de Sucesso:** Linha do tempo destacando países com políticas públicas exitosas nessas áreas.
- **Mídia Recomendada:** Curadoria editorial de músicas, documentários e palestras (TED Talks) relacionadas aos temas.
- **ONGs e Organizações:** Endereços das principais entidades focadas em transparência, combate à pobreza e proteção ambiental.
- **Referências Bibliográficas:** Base acadêmica que sustenta a pesquisa do grupo.

## Tecnologias Utilizadas

O projeto foi construído sem o uso de frameworks complexos, focando em performance, acessibilidade e estilização nativa:
- **HTML5** (Semântico)
- **CSS3** (Variáveis, Grid, Flexbox, Animações e Temas Dark/Light)
- **JavaScript (Vanilla)** (Intersection Observers, Modais, Carrossel)
- Scripts opcionais para servidor local (`server.js` em Node.js e `server.pl` em Perl).
